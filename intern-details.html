Intern Management System – Corporate Internship Project

The Intern Management System is a web-based application developed as part of my corporate internship at Awakeen Studio Pvt. Ltd. The system was designed to streamline the onboarding, tracking, and communication of interns within an organization, covering the entire lifecycle from application to feedback.

The platform features two core modules: a public-facing portal for interns to apply, register, and submit required documents, and an admin dashboard for managing intern details, reviewing applications, scheduling reviews, and issuing certificates. The system also includes email-based communication, Microsoft form integration, and document handling (PDF attachments for certificates).

Key modules include login/registration with validation, categorized internship listings (online/offline, part-time/full-time), admin grids for tracking intern progress, calendar scheduling, a feedback system, and a “guide satisfaction” module to evaluate intern performance.

This hands-on project significantly strengthened my skills in ASP.NET, backend logic design, form handling, and real-world database interactions. I also learned project flow testing, troubleshooting redirection and email issues, and integrating Microsoft Teams and Forms for meeting management.

Technologies Used: ASP.NET, HTML5, CSS3, JavaScript, JQuery, MSSQL
