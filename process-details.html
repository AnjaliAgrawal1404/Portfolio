Thinksense Process Miner – Process Intelligence Tool

Thinksense Process Miner is a web-based analytics tool designed to visualize, analyze, and optimize complex business processes using event logs. Built with business analysts in mind, the tool identifies bottlenecks, inefficiencies, and rework cycles through dynamic data-driven insights. It bridges the gap between data and decisions by converting event data into interactive process maps, statistical charts, and variant flows.

My core contributions were in the development of two critical modules: Raw/Filter Data Viewer and Variant Analysis.

In the Raw/Filter Data Viewer module, I implemented functionality that allows users to view raw event logs, apply filters on specific attributes (like activity or team code), and export filtered datasets to Excel. I also enabled users to save and update filtered views, improving user workflow and reusability of filters across modules such as Process Map and Variant.

In the Variant module, I contributed to building a system for visualizing and analyzing process variants. Features included variant filtering, statistical breakdown of activity durations (min, max, mean), percentage-based insights, and timeline charts that illustrate event sequences. I also enabled export options for complete variant reports, enhancing data portability and reporting.

The overall tool empowers business users to detect high-cost loops, delays, and reworks without deep technical skills. All visual insights are interactive and exportable, including Gantt charts, frequency maps, and variant timelines.

Technology Used: C#, ASP.NET, DevExpress UI Tools, SQL Server
