Café Management System – Complete Reservation & Ordering Platform

The Café Management System is a comprehensive, full-stack web application developed during my industrial internship at GateTouch. The system was built to digitally transform the operations of a modern café by providing users with features like table reservations, food ordering, event booking, banquet hall reservations, and arcade game bookings — all through a centralized platform.

The system consists of two fully functional portals: one for customers and one for administrators. Users can view the menu, gallery, and blog posts, as well as register, log in, place orders, reserve spaces, provide feedback, and make online payments. The admin portal includes complete control over system data, including management of reservations, offers, subscribers, chefs, feedback, menu items, and payments.

Additional features include OTP-based password recovery, integration (including PayPal Payment Gateway) , blog management, newsletter subscriptions, and automated emails for reservation updates. Both UI and UX were carefully designed to ensure responsiveness and ease of use across devices.

This project strengthened my skills in modular development, form validation, backend CRUD operations, and security implementation. It also exposed me to real-time project planning, requirement gathering, and professional communication.

Technologies Used: PHP (OOP), MySQL, HTML5, CSS3, Bootstrap 5, JavaScript, JQuery, AJAX, PayPal API
