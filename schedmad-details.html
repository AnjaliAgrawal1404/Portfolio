Schedmad CRM – SuiteCRM 8 Migration and Enhancement

Schedmad is an internal CRM system used by the company to manage operations and client workflows. During my tenure, I was responsible for migrating this legacy CRM from SuiteCRM 7 to SuiteCRM 8, ensuring complete module compatibility and improved performance.

Post-migration, I developed and packaged custom reports using installable extensions, allowing advanced data analysis through the upgraded system. I also implemented several client-side feature requests, including quick action buttons and customized workflow enhancements.

Additionally, I Dockerized the entire SuiteCRM 8 project for easier deployment and environment consistency, and integrated Elasticsearch to improve search performance across CRM modules. Email integration was also a critical part of the upgrade — I configured both inbound and outbound email settings using Microsoft Outlook OAuth2 authentication.

This project helped me build in-depth skills in system architecture, extension development, Docker configuration, and enterprise-level email and search integration.

Technologies Used: SuiteCRM 7 & 8, PHP, Symfony, Docker, Elasticsearch, OAuth2, Microsoft Outlook Integration
